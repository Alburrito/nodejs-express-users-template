# nodejs-express-users-template

NodeJS template with user management system.

## Installation

1. Clone the repository

```bash
git clone https://github.com/Alburrito/nodejs-express-users-template.git
```

2. Install packages and dependencies

```bash
npm install
```

3. Development mode:

```bash
npm run dev
```

3.5. Production mode

```bash
npm run build
npm start
```
